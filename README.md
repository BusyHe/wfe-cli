# wfe-cli

> Frontend Quick Start Init Project

## Installation

```shell
npm install wfe-cli -g
# or
yarn global add wfe-cli
```

## Getting Started

### Templates

```shell
# get template list
wfe list

# init project
wfe init <template> <projectName>
```

### Repos

```shell
# get repo list
wfe repos list

# use repo
wfe repos use <repo>

# add repo
wfe repos add <reponame> <reposUrl>
```
